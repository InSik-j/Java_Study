문제 5: 도형의 면적과 둘레 계산기 구현하기
1.	기본 요구 사항
    •	Shape라는 추상 클래스를 만들고, 이 클래스에는 getArea()와 getPerimeter()라는 두 개의 추상 메소드를 정의합니다. 이 메소드들은 각각 도형의 면적과 둘레를 계산하여 반환합니다.
2.	서브클래스 구현
    •	Rectangle (직사각형), Circle (원), Triangle (삼각형) 세 개의 서브클래스를 만들고, Shape를 상속받습니다.
        •	Rectangle 클래스는 너비와 높이를 속성으로 가지며, 면적과 둘레를 계산합니다.
        •	Circle 클래스는 반지름을 속성으로 가지며, 면적과 둘레를 계산합니다.
        •	Triangle 클래스는 세 변의 길이를 속성으로 가지며, 면적과 둘레를 계산합니다.
            * 면적 계산은 헤론의 공식을 사용할 수 있습니다.
3.	ShapeCalculator 클래스 구현
    •	ShapeCalculator 클래스에서는 Shape 타입의 리스트를 만들어, 다양한 도형의 인스턴스를 저장합니다.
    •	저장된 각 도형의 면적과 둘레를 계산하여 출력합니다.
        이 때, 모든 도형의 면적과 둘레를 순회하며 출력하는 메소드를 포함합니다.
추가 설명
    •	헤론의 공식: 삼각형의 세 변의 길이가 주어졌을 때 면적을 구하는 공식입니다. 삼각형의 세 변의 길이를 a, b, c라고 할 때, 반둘레 s = (a + b + c) / 2이고, 면적 A는 다음과 같이 계산할 수 있습니다. [ A = \sqrt{s(s-a)(s-b)(s-c)} ]
