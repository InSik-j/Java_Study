* 뤼튼에서 출력한 문제

문제3 : 도형 그리기 프로그램
    1.	도형(Shape) 클래스 생성
        •	Shape는 모든 도형의 상위 클래스입니다.
        •	draw() 메소드는 추상 메소드로, 도형을 그리는 방법을 출력합니다.
    2.	서브클래스 생성
        •	Circle, Square, Triangle 세 가지 서브클래스를 생성합니다.
        •	각 클래스는 Shape를 상속받고 draw() 메소드를 오버라이드하여, 각 도형에 맞는 그리기 메시지를 출력합니다.
    3.	도형 그리기(DrawShapes) 클래스
        •	main 메소드에서 Shape 타입의 배열을 생성하고, 각기 다른 도형의 인스턴스로 채웁니다.
        •	배열을 순회하면서 각 도형의 draw() 메소드를 호출합니다.
